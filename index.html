End Metals

( Info ) :

End Metals is an Addon for the mod Base Metals, it expands it to bring its ores to the End dimension.

It can fit perfectly in skyblock modpacks

This mod can make use of Base Metals (By: DrCyano) http://minecraft.curseforge.com/projects/base-metals This mod can make use of Modern Metals (By: jriwanek) http://minecraft.curseforge.com/projects/modern-metals
